# Sample Rails project with local SQLite solid_cache

Two simple steps to get the app running:

1. Run

```sh
$ bin/setup
```

2. Enable caching in development

```sh
$ bin/rails dev:cache
```

3. Start server

```sh
$ bin/rails server
```
